## Step 6: PDF 원본 비교 (Python 스크립트 사용)

**중요**: 15.9MB PDF 전체를 Read tool로 읽으면 실패합니다. Python 스크립트 사용 필수!

**명령**:
```bash
python scripts_temp/extract_section_from_pdf.py $ARGUMENTS
```

**출력 예시**:
```json
{
  "section_id": "9.8",
  "start_page": 741,
  "pages_extracted": 5,
  "total_chars": 11995,
  "preview": "Section 9.8.\nStairs, Ramps, Handrails and Guards\n..."
}
```

**확인 항목**:
1. **preview 첫 200자를 JSON content와 비교**
   - [ ] 섹션 제목 일치
   - [ ] 첫 subsection (9.x.1) 시작 부분 일치
2. **참조 텍스트 뒤섞임 확인**
   - [ ] "Articles 9.3.1.6." 같은 참조가 별도 블록으로 분리 안 됨
3. **Article 순서 정상**
   - [ ] 9.x.x.1, 9.x.x.2, 9.x.x.3 순서대로 나열

**결과**:
- 모두 OK → Step 7 진행
- 이슈 발견 → `CLAUDE.md` 실수 기록 확인 후 수정
