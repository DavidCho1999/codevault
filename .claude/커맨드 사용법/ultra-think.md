# /ultra-think

**깊은 분석 모드**

복잡한 문제를 여러 관점에서 다차원적으로 분석.

## 사용법
```
/ultra-think "분석할 문제나 질문"
```

## 예시
```
/ultra-think "마이크로서비스로 전환해야 할까?"
/ultra-think "10배 트래픽을 어떻게 처리할까?"
/ultra-think "어떤 기술 스택을 선택할까?"
```

## 분석 관점
- **기술**: 확장성, 성능, 유지보수성
- **비즈니스**: ROI, 시장 경쟁력
- **사용자**: UX, 접근성
- **시스템**: 통합, 의존성

## 출력 내용
- 3-5개 솔루션 제시
- 각 솔루션의 장단점
- 추천 및 근거
- 구현 로드맵
