# /todo

**프로젝트 할일 관리**

프로젝트 루트에 `todos.md` 파일로 할일을 관리.

## 사용법
```
/todo add "작업 설명"           # 할일 추가
/todo add "작업" tomorrow       # 기한 지정 추가
/todo complete 1               # 1번 완료 처리
/todo remove 2                 # 2번 삭제
/todo list                     # 전체 목록
/todo next                     # 다음 할일
/todo undo 1                   # 완료 취소
```

## 기한 지정 예시
- `tomorrow`, `next week`
- `4 days`, `June 9`
- `12-24-2025`
