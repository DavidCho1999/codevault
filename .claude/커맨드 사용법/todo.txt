  사용 방법
  ┌────────────────────────────┬──────────────────────┐
  │           명령어           │         설명         │
  ├────────────────────────────┼──────────────────────┤
  │ /todo add "할 일"          │ 새 할 일 추가        │
  ├────────────────────────────┼──────────────────────┤
  │ /todo add "할 일" tomorrow │ 마감일 포함 추가     │
  ├────────────────────────────┼──────────────────────┤
  │ /todo complete 1           │ 1번 할 일 완료 처리  │
  ├────────────────────────────┼──────────────────────┤
  │ /todo remove 2             │ 2번 할 일 삭제       │
  ├────────────────────────────┼──────────────────────┤
  │ /todo list                 │ 전체 목록 보기       │
  ├────────────────────────────┼──────────────────────┤
  │ /todo undo 1               │ 완료된 항목 되돌리기 │
  ├────────────────────────────┼──────────────────────┤
  │ /todo next                 │ 다음 할 일 보기      │
  ├────────────────────────────┼──────────────────────┤
  │ /todo past due             │ 마감 지난 항목 보기  │
  └────────────────────────────┴──────────────────────┘
