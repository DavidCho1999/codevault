# /explain-code

**코드 분석 및 설명**

코드가 어떻게 동작하는지 분석하고 설명해줌.

## 사용법
```
/explain-code "설명할 코드나 파일"
```

## 예시
```
/explain-code "useAuth hook"
/explain-code "src/lib/parser.ts"
```

## 설명 내용
- 전체 구조 개요
- 함수/클래스 역할
- 알고리즘 설명
- 데이터 흐름
- 에러 처리 방식
- 성능 고려사항

## 언어별 특화 설명
- **JS/TS**: async/await, closure, this 바인딩
- **Python**: 컴프리헨션, 데코레이터, 컨텍스트 매니저
- **Go**: goroutine, channel, 인터페이스
