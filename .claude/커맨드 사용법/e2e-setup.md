# /e2e-setup

**E2E 테스트 환경 설정**

Playwright, Cypress 등 E2E 테스트 환경을 구성.

## 사용법
```
/e2e-setup
/e2e-setup --cypress
/e2e-setup --playwright
/e2e-setup --webdriver
/e2e-setup --puppeteer
/e2e-setup --mobile
```

## 옵션 설명
- `--cypress`: Cypress 설정
- `--playwright`: Playwright 설정
- `--webdriver`: WebDriver 설정
- `--puppeteer`: Puppeteer 설정
- `--mobile`: 모바일 테스트 설정

## 설정 항목
- 프레임워크 설치 및 설정
- 테스트 환경 구성
- Page Object 패턴
- 테스트 데이터 관리
- CI/CD 통합
- 크로스 브라우저 테스트
