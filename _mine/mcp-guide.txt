================================================================================
MCP 서버 용도별 가이드 (Deep Search 결과)
================================================================================

> "MCP = AI를 위한 USB-C 포트" - 다양한 도구/서비스 연결 표준
> 핵심 원칙: 2-3개로 시작, 필요할 때 추가 (많으면 컨텍스트 40% 차지)


================================================================================
[1] 문서/검색 - 모든 사람에게 추천
================================================================================

Context7
--------
용도: 최신 라이브러리/프레임워크 문서 실시간 제공
왜 필수: Claude 지식 오래됨 -> 할루시네이션 방지
설치: claude mcp add context7 -- npx @upstash/context7-mcp
URL: https://github.com/upstash/context7

Perplexity
----------
용도: 실시간 웹 검색 + AI 답변 (출처 포함)
장점: 최신 정보 리서치, 소스 인용
URL: https://www.pulsemcp.com/servers/perplexity

Brave Search
------------
용도: 프라이버시 중심 웹 검색
장점: 독립적인 검색 인덱스, 다양한 검색 연산자 지원

Tavily
------
용도: AI 에이전트 최적화 검색
장점: 팩트 중심, 인용 지원, LLM 친화적 포맷


================================================================================
[2] 개발 도구 - 개발자 필수
================================================================================

GitHub
------
용도: Issue, PR, 커밋, CI/CD 관리
왜 필수: 터미널에서 GitHub 작업 직접 수행
설치: claude mcp add github -- npx @anthropic-ai/github-mcp-server
URL: https://github.com/github/github-mcp-server

Git
---
용도: 로컬 Git 저장소 읽기/검색/조작
장점: 공식 MCP 서버, 안정적

Filesystem
----------
용도: 파일 시스템 안전한 접근
장점: 접근 제어 설정 가능


================================================================================
[3] 데이터베이스 - 백엔드 개발자
================================================================================

PostgreSQL
----------
용도: 자연어 -> SQL 쿼리 변환, 스키마 조회
장점: 테이블 구조 외울 필요 없음

MongoDB
-------
용도: MongoDB 클러스터 연결, 자연어 쿼리
장점: 스키마 인식, NoSQL 작업 자동화

Supabase
--------
용도: Supabase 프로젝트 관리, 분석 쿼리
장점: Auth, Storage, DB 통합 관리


================================================================================
[4] 웹 자동화/테스트 - 프론트엔드/QA
================================================================================

Playwright
----------
용도: 브라우저 자동화, E2E 테스트
장점: 접근성 트리 기반 (스크린샷 불필요), 안정적
URL: https://github.com/playwright-community/mcp-server-playwright

Puppeteer
---------
용도: Chrome 자동화, 웹 스크래핑
장점: 가볍고 빠름


================================================================================
[5] 디자인 -> 코드 - UI/UX 개발자
================================================================================

Figma (Dev Mode)
----------------
용도: Figma 디자인 -> React/Vue 코드 생성
장점: 실제 디자인 구조 기반 코드 (스크린샷 X)
URL: https://github.com/figma/figma-mcp-server

Magic UI
--------
용도: React + Tailwind 컴포넌트 라이브러리
장점: "마퀴 만들어줘" -> 프로덕션 레디 코드


================================================================================
[6] DevOps/인프라 - 플랫폼 엔지니어
================================================================================

Docker
------
용도: 컨테이너 오케스트레이션, 디버깅, 배포 검증

Kubernetes (k8m)
----------------
용도: 멀티 클러스터 K8s 관리, 자연어 명령
장점: 50개 이상 내장 도구
URL: https://github.com/weibaohui/k8m

Terraform
---------
용도: 인프라 코드 관리, 배포 자동화


================================================================================
[7] 에러 추적/모니터링 - 운영팀
================================================================================

Sentry
------
용도: 에러 추적, 성능 모니터링, 자동 분석
장점: AI 근본 원인 분석 (Seer), 수정 추천
설치: claude mcp add --transport http sentry https://mcp.sentry.dev/mcp
URL: https://github.com/getsentry/sentry-mcp


================================================================================
[8] 생산성/협업 - 팀 작업
================================================================================

Notion
------
용도: 문서/DB/태스크 관리
장점: AI가 Notion 검색, 페이지 생성, DB 업데이트
URL: https://developers.notion.com/docs/mcp

Slack
-----
용도: 메시지 전송, 검색, 채널 관리
장점: 버그 리포트 -> Issue 자동 생성 워크플로우

Linear
------
용도: 이슈 트래킹, 프로젝트 관리


================================================================================
[9] 데이터 사이언스 - 분석가/연구자
================================================================================

Jupyter
-------
용도: Jupyter Notebook 실시간 제어, 코드 실행
장점: 데이터 분석, 시각화, 모델 학습 자동화
사용예: 데이터 클리닝, seaborn/matplotlib 차트, sklearn 모델
URL: https://github.com/datalayer/jupyter-mcp-server

[10] 메모리/지식 관리 - 장기 컨텍스트
================================================================================

Memory (Knowledge Graph)
------------------------
용도: 대화 간 기억 유지, 지식 그래프
구조: 엔티티 -> 관계 -> 관찰
장점: 세션 넘어서 정보 기억
URL: https://github.com/modelcontextprotocol/servers/tree/main/src/memory

Basic Memory
------------
용도: 로컬 Markdown 기반 지식 관리
장점: 데이터 로컬 보관, 양방향 동기화


================================================================================

Sequential Thinking
-------------------
용도: 복잡한 문제 단계별 분석

장점: 마이크로서비스 설계, 트레이드오프 분석
