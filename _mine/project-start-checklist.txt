================================================================================
[실전 예시] 새 프로젝트 시작
================================================================================

1. 폴더 생성
   mkdir my-project && cd my-project

2. CLAUDE.md 생성
   "이 프로젝트용 CLAUDE.md 만들어줘. 
   [프로젝트 설명]"

3. Plan Mode로 계획
   "Plan Mode로 이 프로젝트 계획해줘.
   바로 코딩하지 말고 접근법 먼저."

4. 승인 후 작은 단계로 구현
   "1단계: 프로젝트 구조부터 잡아줘"

5. 각 단계마다 검증
   "테스트해보자. 잘 작동하는지 확인해줘"

6. 실수 발생 시 기록
   "이 실수 CLAUDE.md에 추가해줘"

================================================================================
[하지 말아야 할 것]
================================================================================

1. 계획 없이 바로 코딩 요청
   X "전체 앱 만들어줘"
   O "Plan Mode로 계획 먼저"

2. 한 번에 큰 결과물 요청
   X "완성된 웹사이트"
   O "로그인 페이지만 먼저"

3. 컨텍스트 무시
   X 계속 대화만
   O /clear 자주

4. 검증 없이 진행
   X 코드만 받고 끝
   O "테스트해보자"

5. 실수 무시
   X 그냥 넘어감
   O CLAUDE.md에 기록
